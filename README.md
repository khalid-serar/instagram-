# Description
This is a Django Instagram clone application where, a user can sign up to have a profile in the application. The user can also view comments, comment, like and post on the application.

## Technologies used

Python 3.8.10
HTML
Django 3.2.5
Bootstrap 3
Heroku
Postgresql
